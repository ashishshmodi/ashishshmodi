# Hi 👋, I'm Ashish Modi

### Senior Data Engineer focused on Azure, AI, and modern analytics platforms

I build data products and platform capabilities that make analytics systems more reliable, scalable, and intelligent. My day-to-day work sits at the intersection of data engineering, platform design, and applied AI, with a strong focus on Azure, Microsoft Fabric, People Analytics, and developer productivity.

I am especially interested in using AI agents and automation to simplify hard engineering problems such as synthetic data generation, data masking, pipeline troubleshooting, deployment workflows, and end-to-end analytics enablement.

## What I work on

- Designing AI-powered data platforms for People Analytics on Azure and Microsoft Fabric
- Building utilities for synthetic data generation, data privacy, and realistic test-data workflows
- Automating deployment and operational workflows for analytics platforms
- Improving data pipeline reliability, performance, and maintainability
- Translating complex business problems into scalable engineering solutions

## Current focus

- AI agents for data engineering and platform operations
- LLM-powered workflows for data generation, validation, and remediation
- Advanced system design and distributed data platforms
- Open-source accelerators that improve developer and analyst productivity

## Areas I enjoy discussing

- Azure Data Engineering
- Microsoft Fabric and analytics platform architecture
- AI agents and practical LLM integration
- Data pipelines, BI, and analytics engineering
- Performance optimization and distributed systems
- .NET and Python for platform tooling

## Selected public work

- Prompt2Data: an AI-driven synthetic data generator agent that turns natural-language prompts into realistic, validated, analysis-ready datasets
- Fabric-focused engineering content covering data engineering patterns, agent workflows, and platform automation
- Open-source contributions around analytics accelerators and operational tooling
